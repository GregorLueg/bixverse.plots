# Bixverse Fill Scale (Discrete)

Bixverse Fill Scale (Discrete)

## Usage

``` r
scale_fill_bx(palette = "main", reverse = FALSE, ...)
```

## Arguments

- palette:

  Palette name (default: "main")

- reverse:

  Reverse colors? (default: FALSE)

- ...:

  Additional arguments passed to scale_fill_manual

## Value

A ggplot2 scale object
