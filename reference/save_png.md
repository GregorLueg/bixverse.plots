# Helper function to save various plot types to PNG

Helper function to save various plot types to PNG

## Usage

``` r
save_png(plot, f_path, plot_params)
```

## Arguments

- plot:

  The plot you want to save to disk.

- f_path:

  File path. The path to which you want to save the PNG

- plot_params:

  List. Output of
  [`params_plots()`](https://gregorlueg.github.io/bixverse.plots/reference/params_plots.md).

## Value

Saves the PNG to disk.
