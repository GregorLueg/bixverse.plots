# Set Bixverse Theme as Default

Sets the Bixverse theme as the default for all ggplot2 plots in the
session

## Usage

``` r
set_bx_theme(base_size = 12, base_family = "Helvetica")
```

## Arguments

- base_size:

  Base font size (default: 12)

- base_family:

  Base font family (default: "Arial")

## Value

NULL (sets theme invisibly)

## Examples

``` r
if (FALSE) { # \dontrun{
 set_bx_theme()
} # }
```
