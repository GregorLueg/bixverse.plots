# Bixverse Color Scale (Discrete)

Bixverse Color Scale (Discrete)

## Usage

``` r
scale_color_bx(palette = "main", reverse = FALSE, ...)
```

## Arguments

- palette:

  Palette name (default: "main")

- reverse:

  Reverse colors? (default: FALSE)

- ...:

  Additional arguments passed to scale_color_manual

## Value

A ggplot2 scale object
